당신의 머신을 공격합니다.NEW

Claude Code · Cursor · VS Code · Windsurf — 모두 MCP 서버를 통해 외부 도구를 호출합니다. 그 설정 한 줄에 시크릿이 박혀 있고, 권한이 열려 있고, 프롬프트 인젝션 페이로드가 숨어 있어도 기존 보안 도구는 보지 못합니다.

한 줄로 머신 전체 점검

npx papurion-mcp-scan 한 줄이면 IDE의 MCP 설정을 자동으로 찾아 6가지 보안 검사를 수행합니다. 네트워크 호출 없이 정적 분석이 기본이고, 시크릿 값은 절대 출력되지 않습니다.

  • 프롬프트 인젝션 — 도구 설명·env에 숨은 인젝션 / 제로폭 문자 / base64 우회
  • 시크릿 노출 — 하드코딩된 API 키, 토큰, DB 연결 문자열
  • 권한 감사 — 루트/홈 디렉토리, Docker 소켓, sudo, rm -rf
  • Tool Poisoning — @modelcontextprotocol/* typosquat 탐지
  • 전송 보안 — HTTP 미암호화, 0.0.0.0 바인딩, 인증 누락
  • Shadow MCP — 조직 allowlist에 없는 비인가 서버

How it works

자산 등록부터 리포트까지,
한 사이클로 흐릅니다.

각 단계는 자동화되어 있고, 사람이 개입할 지점은 명확히 분리되어 있습니다. CI/CD에 연동하면 매 PR마다 같은 사이클이 돌아갑니다.

자산 등록

GitHub repo 또는 web URL을 대시보드에 등록합니다.

스캔 실행

격리된 Fly Machine 컨테이너에서 SAST·SCA·DAST·Secret이 병렬 실행됩니다.

AI 분석

다중 AI가 finding을 검토하고 도달 가능성과 PoC를 생성합니다.

Delta 추적

fingerprint로 New · Persistent · Resolved · Reopened를 자동 분류합니다.

알림 · 자동 수정

Slack으로 즉시 알리고, Pro 이상은 AI 수정 PR이 자동 생성됩니다.

리포트

OWASP·CWE·ISO 27001 매핑이 포함된 PDF 컴플라이언스 리포트로 마무리.
